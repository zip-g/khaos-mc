# Create：回転力・歯車システム

Create MODでは、多くの機械が**回転力（Rotation Power）**によって動作します。  
このページでは、サーバー参加者（クライアント側）が最低限理解しておくべき  
**回転力の考え方・歯車の使い方・注意点**をまとめています。

## 回転力とは

Createにおける回転力は、以下の3要素で管理されています。

- 回転速度（RPM）
- 回転方向（正転 / 逆転）
- トルク（負荷に耐える力）

機械は**速度が足りない**、または**負荷が重すぎる**と停止します。

## 回転の発生源

以下は、サーバーで一般的に使用可能な回転源です。

- 水車
- 風車
- 蒸気エンジン
- 手回しクランク（初期用）

回転源は直接シャフトや歯車に接続することで、周囲の機械へ動力を伝達します。

## シャフト（軸）

シャフトは回転力を**直線方向**に伝えるための基本部品です。

- 回転速度は変化しない
- 長く伸ばすほど負荷が増える
- 直線専用（方向転換不可）

方向を変えたい場合は歯車が必要です。

## 歯車の基本

歯車は回転を**伝える・方向を変える・速度を変える**ために使います。

### 小歯車と大歯車

歯車にはサイズ差があります。

- 小歯車 → 大歯車  
  → 回転速度が遅くなる / トルクが増える  

- 大歯車 → 小歯車  
  → 回転速度が速くなる / トルクが減る  

高速化しすぎると機械が破損・停止するため注意してください。

## 回転方向について

歯車を1つ噛ませるごとに、回転方向は**反転**します。

例：

- 歯車1枚 → 逆回転
- 歯車2枚 → 元の回転方向

方向が合わない場合は、歯車の枚数で調整してください。

## 機械が動かないときのチェックポイント

以下はよくある原因です。

- 回転速度が不足している
- 速度が速すぎて過負荷になっている
- 歯車の噛み合わせが正しくない
- 回転方向が逆で動作条件を満たしていない

Createの機械は**見た目が繋がっていても、論理的に成立していないと動きません**。

## サーバーでの注意事項

- 極端な高速回転装置はラグや制限対象になる場合があります
- 無限増速構成は禁止されることがあります
- 公共設備に接続する場合は事前に確認してください

Createの機械は、**考えて組むほど強くなる**のが魅力です。
