# Create

Create は「回転力」を用いて機械や装置を構築し、  
生産・加工・運搬などを自動化できる大型 MOD です。

本サーバーでは Create を主要コンテンツのひとつとして採用しており、  
拠点運用や資源確保に大きく関わります。

## Create の特徴

- 電力（RF / FE）を使用しない
- 回転方向・回転速度が重要
- 機構の組み方次第で性能が大きく変化する
- 視覚的に動きが分かりやすい

他の工業系 MOD とは考え方が異なるため、  
基本概念の理解が前提となります。

## 回転力について

Create では、すべての機械が **回転力** によって動作します。

回転力には以下の要素があります。

- 回転速度（RPM）
- 回転方向
- ストレス値（負荷）

回転力は歯車・シャフト・ベルトなどを通じて伝達されます。

## 回転速度（RPM）

回転速度は、装置の動作速度に直接影響します。

- 高速回転：処理は速いが負荷が増加
- 低速回転：安定するが処理が遅い

歯車の大きさを変えることで回転速度を変換できます。

- 小 → 大：減速
- 大 → 小：加速

## 回転方向

Create の機械は **回転方向** も重要です。

- 歯車を直接接続すると回転方向が反転する
- シャフトやベルトでは方向が維持される

回転方向が逆の場合、機械が動作しないことがあります。

## ストレス値（負荷）

各 Create の機械は **ストレス値** を消費します。

- 水車・風車などの動力源には最大出力がある
- 消費ストレスが出力を超えると機械が停止する

Jade を使用すると、  
各機械のストレス消費量を確認できます。

## 動力源

代表的な動力源は以下の通りです。

- 水車
- 大型水車
- 風車
- エンジン系ブロック

序盤は水車、中盤以降は大型水車や複数動力源の併用が一般的です。

## 自動化について

Create では以下のような自動化が可能です。

- 鉱石・素材の加工
- 作物収穫
- クラフト工程の自動化
- 搬送・仕分け

装置の規模が大きくなるほど、  
回転速度・ストレス管理が重要になります。

## サーバーでの注意点

本サーバーでは、以下の点に注意してください。

- 不要に高速な回転装置を常時稼働させない
- 無限ループ構造を作らない
- 大規模装置は分割・停止機構を設ける
- チャンク外でも動作し続ける構成に注意する

過度な負荷が確認された場合、  
装置の修正をお願いすることがあります。
