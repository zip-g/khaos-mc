# サーバー概要

本サーバーは **Fabric 1.20.1** を基盤とした、  
探索・戦闘・自動化（Create）を中心に拡張された MOD サーバーです。

バニラのゲーム性を尊重しつつ、大型MODによるコンテンツ追加と QoL 改善を目的としています。

| Minecraft バージョン | **1.20.1** |
| MOD ローダー | **Fabric** |

## 採用 MOD 方針

本サーバーでは以下を重視しています。

- 探索・ダンジョン要素の拡張
- 戦闘体験の刷新（Better Combat 系）
- 自動化・機械要素（Create）
- 視覚・操作性の向上（QoL MOD）

単なる要素追加ではなく、  
**MOD 同士の組み合わせによる遊びの幅**を重視しています。


## Create について

Create は本サーバーにおける **主要MODのひとつ**です。

- 回転力を用いた機械構築
- 自動化による資源生産
- 建築・装置設計の自由度向上

::: warning 注意
Create の装置は構成次第でサーバー負荷が高くなる場合があります。  
過度な高速回転装置や無限ループ機構には制限があります。
:::

詳細は Create 専用ページを参照してください。


## クライアント向けリファレンスについて

本ドキュメントでは以下を目的としています。

- クライアント側で **理解が必要な MOD の解説**
- 操作方法・挙動・注意点の明文化
- 初参加者が迷わないための情報整理

サーバー内部処理や API / ライブラリ系 MOD は、  
原則として解説対象外としています。
