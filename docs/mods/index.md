# 必須MOD一覧

本ページでは、本サーバーに参加するために  
**クライアント側で導入が必須、または仕様理解が前提となる MOD** をまとめています。

ここに記載されている MOD を未導入の場合、  
接続できない、または正常にプレイできない可能性があります。

## 必須MODについて

本サーバーの MOD は以下のように分類されています。

- **必須**：未導入だと接続不可、または致命的な不具合が発生
- **準必須**：未導入でも接続可能だが、ゲーム体験が大きく損なわれる
- **任意**：導入推奨。操作性や視認性が向上する

このページでは **必須・準必須 MOD** のみを扱います。

## 基幹・システム系

### Create

回転力を用いて機械や自動化を行う大型MODです。  
本サーバーでは主要コンテンツのひとつとして採用されています。

- 自動化・生産設備
- 回転力・ストレス管理
- 機械構築による建築要素

詳細は Create 専用ページを参照してください。

### EMI（Enhanced Minecraft Interface）

レシピ検索・クラフト補助を行う MOD です。

導入されているアドオン：
- emi_enchanting
- emi_loot
- emi_ores
- emitrades

クラフト・エンチャント・戦利品確認に必須です。

### Jade

ブロックやエンティティの情報を HUD 表示する MOD です。

- ブロックの状態確認
- 作業台・機械の情報表示
- MOD ブロックの挙動把握

Create や各種 MOD の理解に不可欠です。

## 装備・インベントリ関連

### Trinkets / Accessories

装備スロットを拡張する基盤 MOD です。

- アクセサリー装備
- バックパック・特殊装備対応
- 他 MOD の前提要素

本サーバーでは多数の MOD が依存しています。

### Traveler’s Backpack

背負うタイプのバックパックを追加します。

- 大容量インベントリ
- タンク機能（液体）
- 寝袋・クラフト補助

長距離探索を前提とするため、準必須扱いです。

### Sophisticated Storage

高機能なチェスト・ストレージを追加します。

- フィルター機能
- 自動入出力
- アップグレード拡張

拠点運用において前提となる仕様が多く存在します。

## マップ・移動

### Xaero’s Minimap / World Map

ミニマップおよび全体マップを表示する MOD です。

- 地形把握
- デス地点確認
- ウェイポイント管理

探索・復帰の効率に直結します。

### Waystones

ワープ用のウェイストーンを追加します。

- 拠点間移動
- 探索拠点の共有
- 長距離移動の負担軽減

サーバー内移動の基盤として採用されています。

## 戦闘・操作系（準必須）

### Better Combat

戦闘システムを刷新する MOD です。

- 攻撃モーションの追加
- 武器ごとの挙動変化
- 攻撃タイミングの概念

バニラと操作感が異なるため、理解が必須です。

### Combat Roll

回避行動（ローリング）を追加します。

- スタミナ消費
- 無敵時間の存在
- キー割り当て必須

戦闘バランスに影響します。
