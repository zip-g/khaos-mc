## Things

### 概要

**Things** は Minecraft に新しい装飾アイテムや小物、インタラクティブ要素を追加する Fabric Mod です。  
Vanilla の世界観を維持しつつ、生活感や建築の幅を広げることができます。

### 主な機能

#### 新規アイテム

- 雑貨や小物の追加  
- 家具や装飾のバリエーションを拡張  
- クラフト可能で Vanilla 材料を使用するアイテム多数  
- Redstone と組み合わせ可能なインタラクティブアイテムあり  

### 追加アイテム一覧(抜粋)

| アイテム名 | 説明 |
|------------|------|
| Displacement Tome | テレポート用の魔法書。最大8地点にワープ可能（エンダーパール消費） |
| Bater Wucket | 無限の水バケツ（使い切りなしの水源） |
| Ender Pouch | ポータブルエンダーチェスト（キー操作で開く） |
| Enchanted Wax Gland | 水中浮遊・泳ぎ速度アップ効果を付与（他アイテムと組み合わせて溶岩でも機能） |
| Infinite Water Bucket (Bater Wucket) | 無限水源（道具カテゴリ） |
| Ender Pouch | エンダーチェストを携帯（アイテム） |
| Gleaming Ore Book | Patchouli ガイド本を作成するときに必要な素材（ガイド本用） |
| Guide Book (Patchouli) | Things Mod のアイテム・レシピ集（Patchouli 用コンテンツ） |
| Trinket-style items | 装備に付けることでプレイヤー能力を底上げする小物各種（Examples: Wax Gland など） |

#### インタラクティブ要素

- 一部の家具は開閉やオン／オフ操作が可能  
- Redstone 信号と連動させることもでき、装飾だけでなく実用的にも使える

### 補足

Things は建築や拠点装飾を強化するための Mod です。  
単体でも便利ですが、他の家具・装飾系 Mod と組み合わせることでさらに多彩な表現が可能になります。
