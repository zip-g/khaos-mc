# Sophisticated Storage Mod（概要）

**Sophisticated Storage** は、Minecraft の **ストレージシステムを大幅に拡張する** MOD です。  
バレル、チェスト、シュルカーボックスを強化し、**カラー変更・アップグレード・自動化機能** など多彩な機能が追加されます。

## 🎒 基本機能

- **既存のストレージブロックを強化可能**  
  バニラのバレル、チェスト、シュルカーボックスに対して機能性を追加。  
- **カラー・見た目のカスタマイズ**  
  染料を使って本体やアクセントの色を変更でき、整理と見た目を両立。

## 🧱 ストレージの種類

| ストレージ | 特徴 |
|------------|------|
| バレル | 標準の木材バリエーションに対応。階層（ティア）アップグレードで容量・スロット数増加。 |
| リミテッドバレル | 1〜4 種類のアイテムを格納可能。スタックサイズ大幅増加。 |
| チェスト | 階層アップで容量増加。 |
| シュルカーボックス | カバン感覚で持ち運び可能。階層アップで容量増加。 |

## 🔧 階層アップグレード

| ティア | 特徴 |
|--------|------|
| 銅（Copper） | 初期アップグレード。基本容量増加。 |
| 鉄（Iron） | 中級容量・アップグレードスロット増加。 |
| 金（Gold） | さらに容量とスロットが増加。 |
| ダイヤモンド（Diamond） | 高級ストレージ。アップグレードスロット多数。 |
| ネザーライト（Netherite） | 最大容量・最大スロット数。 |

## ⚙️ 機能アップグレード一覧

| アップグレード | 機能説明 |
|----------------|----------|
| Pickup Upgrade | 地面のアイテムを自動回収。選択フィルター付き。 |
| Magnet Upgrade | 周囲のアイテムや XP オーブを吸引。 |
| Compacting Upgrade | アイテムを自動圧縮（例：インゴット→ブロック）。 |
| Void Upgrade | 不要アイテムを自動破棄。 |
| Smelting / Smoking / Blasting Upgrades | ストレージ内で自動加工（かまど系処理）。 |
| Crafting / Stone Cutter Upgrades | ストレージ内でクラフト・石切り処理が可能。 |
| Jukebox Upgrade | 音楽ディスクを再生可能。 |
| Feeding Upgrade | プレイヤー周囲に自動で食料を供給。 |
| Filter Upgrade | 入出力を細かく制御するフィルター機能。 |

## 🕹️ ストレージコントローラー

- 複数のストレージを単一インターフェイスで管理できるハブブロック。  
- 近くのストレージを自動で分類・整理。  
- 右クリック操作で一致するアイテムを一括投入可能。

## 📦 アイテム投入方法

- **直接操作**：コントローラーを右クリックして投入。  
- **ホッパー連携**：ホッパーで自動で供給可能。  
- **ホッパーアップグレード**：バレル内で利用可（条件あり）。

## 🧾 まとめ

Sophisticated Storage は、Minecraft のストレージ管理を **単なる保管以上の機能へ進化** させる MOD です。  
大量アイテム整理、自動化、加工機能など、**ベース構築や大型拠点運用に最適** な拡張要素を提供します。
