# Transmog の使い方ガイド

Transmog は、アイテムの性能や挙動を変えずに、見た目だけを他のアイテムに変えることができる MOD です。
防具・武器・道具・その他ほぼ全てのアイテムに対応し、MOD で追加されたアイテムにも適用できます。

## 概要

Transmog はプレイヤーが所持しているアイテムを、別のアイテムの見た目に変更することができます。
変更は見た目のみで、元のアイテムの性能やステータスには一切影響しません。

- 防具・武器・道具などほぼ全てのアイテムに対応
- MOD 由来のアイテムも見た目変更可能
- 見た目変更によって機能や性能は変わりません

## 追加されるアイテム

### Void Fragment

Transmogrification Table の材料として使用します。
- Table のクラフト用に消費
- 見た目スロットに置くと対象アイテムを透明にすることが可能

### Transmogrification Table

Transmog の中心となるブロックです。
設置して右クリックすると UI が開き、見た目変更を行えます。

## 変身の手順

1. Transmogrification Table を設置し右クリックして UI を開く
2. 左端のスロットに燃料（Amethyst Shard）をセット
3. 中央のスロットに見た目変更したいアイテムをセット
4. 右端のスロットに見た目として使いたいアイテムをセット（消費されません）
5. 条件を満たしたら変身を実行し、対象アイテムの見た目が変更される

## 見た目（Appearance）のルール

通常の見た目変更では、ほとんどのアイテムは Appearance スロットに入れたアイテムのモデル・テクスチャ通りに変わります。

既に Transmogrified なアイテムは、既存の見た目をベースに変更されます。

Void Fragment を Appearance スロットに置くと、対象アイテムが完全に見えない状態（透明）になります。

## 見た目の解除

見た目変更を解除したい場合は、Appearance スロットを空にした状態で対象アイテムを Table に入れることで元の見た目に戻ります。

## 注意点

- 見た目変更のみで性能には影響しません
- 燃料となる Amethyst Shard は操作ごとに消費
- MOD 由来のアイテムにも適用可能で、見た目の統一やコーディネート用途に便利

## まとめ

Transmog は装備やアイテムのビジュアル体験を自由に楽しむための MOD です。
性能を犠牲にせず、お気に入りの見た目でプレイすることができます。
