# Fabric Seasons Mod 詳細解説

## Mod概要
**Fabric Seasons** はMinecraftにリアルな四季（春・夏・秋・冬）を導入するModです。  
季節の変化に応じて環境やゲームプレイに影響を与えることで、より戦略性のあるサバイバル体験を提供します。  
このModは単独でも動作しますが、DehydrationやFrostiful、Scorchfulなどの環境系Modと組み合わせると、体温・水分管理などの要素と連動し、さらにリアルなサバイバルが楽しめます。

### 主な特徴
- 世界全体に季節の概念を導入（季節ごとの天候や気温の変化）。
- バイオームや作物、動物の挙動に季節が影響。
- HUDや環境表現で季節を視覚的に確認可能。
- MODの設定で季節サイクルの速度や気候変化の強度を調整可能。

## 季節ごとの効果

### 春 (Spring)
**概要:**  
春は再生と成長の季節。作物の生長が早まり、温暖な気候でプレイヤーに負担が少ない時期です。  

**効果:**  
- 作物の成長速度増加（通常より+20〜30%程度）。  
- 植物のドロップ率上昇（花、果実など）。  
- 雨の発生頻度が増えるが、暑さや寒さのペナルティは少ない。  
- 動物の繁殖速度増加（ウサギや家畜などがより早く繁殖）。  
- 水分消費（Dehydration Mod使用時）が標準値。

### 夏 (Summer)
**概要:**  
夏は高温・乾燥の季節。探索や活動に影響を与える暑さ要素が増加します。  

**効果:**  
- 気温上昇による水分消費増加（Dehydration Modと連動）。  
- 草や作物の枯渇リスクが増える（特に水源が少ない地域）。  
- 高温地域では体温上昇（Scorchful Mod使用時）によるダメージリスク。  
- 雨は少なめで乾燥しやすい。  
- 一部のモブ（砂漠系モブなど）が活発化。

### 秋 (Autumn / Fall)
**概要:**  
秋は収穫と準備の季節。徐々に気温が下がり、冬への移行が始まります。  

**効果:**  
- 作物や木の果実の収穫時期（ドロップ量増加）。  
- 葉が落ちることで視界が変化し、景観が季節感を演出。  
- 気温が徐々に低下し、寒冷地では体温管理が重要になる（Frostiful Modとの連動）。  
- 動物の活動がやや減少し、繁殖率低下。  
- 雨の発生頻度は標準。

### 冬 (Winter)
**概要:**  
冬は寒冷・厳冬の季節。探索や戦闘、サバイバルにおける難易度が最も高い時期です。  

**効果:**  
- 気温が低下し、Frostiful Mod使用時は体温低下による移動速度・採掘速度低下や凍傷リスク。  
- 作物の成長が遅くなる、または完全に停止。  
- 雪や氷の発生により地形が変化（川が凍結、地面が滑るなど）。  
- 水分消費が抑制される傾向（寒冷地では発汗が少ないため）。  
- 一部のモブ（雪原系モブなど）が活発化。  
- 暖房アイテムや防寒装備の重要性が増す。

## Mod間連携
- **Dehydration:** 夏は水分消費増、冬は消費抑制。  
- **Frostiful / Scorchful:** 冬は寒さによる体温低下、夏は高温による体温上昇に直結。  
- **EnvironmentZ / TerraBlender:** 季節による天候や植生の変化と組み合わせることで、よりリアルな自然環境を表現可能。  

## ゲームプレイへの影響
- 季節に応じて**探索・採取・戦闘戦略**が変化。  
- 作物や動物の管理を意識した農業プレイが可能。  
- 高温・低温の環境適応が必要になり、防具や装備の選択が重要。  
- 季節の変化によって、建築や拠点設計にも戦略性が加わる。
