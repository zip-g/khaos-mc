# Sortilege

Sortilege は Minecraft の魔法・ポーション・経験値システムを拡張する Mod です。  
Vanilla の魔法要素を活かしつつ、より多彩で便利な機能を追加します。

## 魔法杖（Magic Staves）

- 木〜ネザライトまでの素材で魔法杖を追加  
- ビームを放ち、敵に魔法ダメージを与える  
- 射程やダメージは素材によって変化  
- 使用後クールダウンが発生  
- 魔法杖専用の効果や経験値消費設定が可能

## ポーション＆醸造の改善

- 解毒薬などの新しいポーションを追加  
- ポーションのスタック数最大8、飲む時間短縮  
- ポーション飲用後のクールダウン 10 秒  
- Cauldron や Campfire を使った醸造処理が可能  
- データパック対応で柔軟に変更可能

## 経験値関連

- 経験値操作の改善とバランス調整  
- 経験値システムの設定を全面サポート（オフ可／最大レベル制限）  
- 魔法杖と経験値を連動させることが可能

## 補足

Sortilege は魔法・ポーション・経験値のシステムを統合的に改善し、Vanilla の世界に自然な“魔法感”を付与する Mod です。
