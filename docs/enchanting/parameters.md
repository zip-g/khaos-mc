# エンチャント内部パラメータ完全解説

Zenith のエンチャントシステムは、以下の**4つの内部パラメータ**で結果が決まります。  
これを理解することで、エンチャント環境を自在に制御できます。

## 1. Eterna（エテルナ）

- **役割**：エンチャント可能レベルの上限を決定
- **効果**
  - 高いほど高レベルエンチャントが出やすくなる
  - 低いとレベル30でも強力なエンチャントが出にくい
- **本棚配置のポイント**
  - 段数・距離で加算される
  - 過剰に積むとQuantaやArcanaとのバランスが崩れる

## 2. Quanta（クアンタ）

- **役割**：エンチャント結果の「ブレ幅」を制御
- **効果**
  - 高い → ギャンブル性が強くなる（強力なものも弱いものも出やすい）
  - 低い → 安定して予想範囲内の結果が出る
- **本棚配置のポイント**
  - 特定の棚や補助ブロックで上昇/低下可能
  - 安定構成では極力低めに設定

## 3. Arcana（アルカナ）

- **役割**：エンチャント候補の質やレアリティに影響
- **効果**
  - 高い → 高レア・強力エンチャントの出現率が上がる
  - 低い → 出現候補が限定され、凡庸な結果が増える
- **本棚配置のポイント**
  - Arcana 特化棚を多めに配置
  - 安定構成では中程度、レア狙いは高め

## 4. Rectification（レクティフィケーション）

- **役割**：不要エンチャントや装備に適さない能力を排除
- **効果**
  - 高い → 特定の装備向けに最適化されたエンチャントが付きやすい
  - 低い → 無駄なエンチャントや弱体化のリスクあり
- **本棚配置のポイント**
  - Rectification 系棚や補助ブロックで強化
  - ビルド特化装備では最重要

## パラメータ相互作用

- これら4つは単独で動作するのではなく、**同時に作用**します。
- 例：
  - 高Eterna + 高Arcana → 強力でレアなエンチャントが出やすい
  - 高Quanta + 低Rectification → ギャンブル性はあるが無駄エンチャントも出やすい
- 目標に応じてバランスを取ることが重要です。

## パラメータまとめ表

| パラメータ        | 高めの効果                       | 低めの効果                    | 使いどころ                       |
|-----------------|--------------------------------|------------------------------|--------------------------------|
| Eterna          | 高レベルエンチャント出やすい        | 強力エンチャントが出にくい       | 高レベル装備狙い               |
| Quanta          | ギャンブル性大、ブレ幅広い         | 安定して狙った範囲の結果に収まる | 厳選・安定運用                 |
| Arcana          | レア・強力エンチャント増加         | 候補が限定され凡庸な結果に       | レア狙い、強化装備             |
| Rectification   | 不要能力排除、最適化               | 無駄なエンチャントが付くことも    | ビルド特化装備、厳選向け       |
