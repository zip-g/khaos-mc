# 本棚構成の具体例

このページでは、Zenith（Apothic Enchanting準拠）の  
**実用的な本棚構成例**を用途別にまとめます。

画像は使用せず、配置イメージは文章で説明します。

前提条件：
- エンチャントテーブルを中心に配置
- 有効範囲内（通常の本棚判定範囲）
- 基本は左右対称配置
- 本棚・補助ブロックはサーバー設定準拠

## 構成例1：安定厳選構成（初心者〜中級者向け）

### 目的
- 失敗を減らす
- 狙った系統のエンチャントを付ける
- 経験値消費を抑える

### パラメータ傾向
- Eterna：中〜高
- Arcana：中
- Quanta：低
- Rectification：高

### 構成イメージ
- エンチャントテーブルの周囲1〜2段に本棚を配置
- Quantaを上げる棚は最小限
- Rectificationを上げる棚・補助ブロックを優先
- Eternaは必要値を超えないよう調整

### 特徴
- 結果が安定する
- 厳選しやすい
- 高レベルやレア付与は控えめ

## 構成例2：高レベル・レア狙い構成（中級者以上）

### 目的
- 高レベルエンチャントを狙う
- Zenith独自エンチャントの出現率を上げる
- 最終装備候補の作成

### パラメータ傾向
- Eterna：高
- Arcana：高
- Quanta：中
- Rectification：中

### 構成イメージ
- エンチャントテーブルを中心に2〜3段構成
- Arcana上昇系の棚を多めに配置
- Quantaは上げすぎないよう意識
- Rectification不足は妥協点を決める

### 特徴
- 強力な結果が出やすい
- 失敗時の消費が重い
- ある程度の割り切りが必要

## 構成例3：ギャンブル構成（上級者向け）

### 目的
- 最大レベル・最大効果を狙う
- ビルド特化装備の最終強化
- 一発逆転狙い

### パラメータ傾向
- Eterna：最大
- Arcana：最大
- Quanta：高
- Rectification：低〜中

### 構成イメージ
- 本棚を最大数近くまで配置
- Quanta上昇系の棚を意図的に採用
- Rectificationは最低限のみ確保
- 結果のブレを許容する前提で使用

### 特徴
- 成功時は最強クラス
- 失敗率が非常に高い
- 通常運用は非推奨

## よくある構成ミス

- 見た目優先で非対称配置にする
- Quanta系の棚を無自覚に積みすぎる
- Eterna過剰で制御不能になる
- Rectification不足のまま高レベルを狙う

本棚構成は「多ければ強い」ではなく  
**目的に合わせた調整**が重要です。

## どの構成を使うべきか

- 普段使い装備 → 安定厳選構成
- 最終装備候補 → 高レベル構成
- ロマン・趣味 → ギャンブル構成

用途ごとに  
**エンチャント環境を切り替える運用**を推奨します。
